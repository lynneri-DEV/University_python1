### О библиотеке Pandas  
**Pandas** — это популярная библиотека языка Python для работы с данными 📊.
Её используют для анализа, обработки и хранения таблиц.  
**Pandas позволяет:**
* читать данные из файлов (Excel, CSV, JSON и др.)
* работать с таблицами
* фильтровать данные
* сортировать
* анализировать информацию
* очищать данные
* делать статистику

1. ### Исследование данных  
- Загружаем файл `df = pd.read_csv('orders.csv')`  

- `pd.read_csv('orders.csv')` - выполняет загрузку данных из файла формата `CSV` (`Comma-Separated Values`) и преобразует их в структуру `DataFrame`.  
- Метод `head()` выводит первые строки таблицы (по умолчанию — 5).
- Метод `to_datetime()` преобразует данные столбца `order_date` в формат даты и времени (`datetime`).
- Метод `info()` выводит общую информацию о `DataFrame`.
- Метод `isnull()` проверяет наличие пропущенных значений (`NaN`) в таблице.
- `df.isnull().sum()` - Комбинация методов выполняет подсчёт пропущенных значений в каждом столбце.

2. ### Очистка данных
* Логическая индексация `df[...]` — фильтрует таблицу, оставляя только те строки, которые соответствуют заданному условию (например, цена выше нуля).
* Оператор `&` — связывает несколько условий фильтрации в одно (логическое «И»), требуя выполнения всех условий одновременно.
* `.copy()` — создает полную независимую копию части таблицы, чтобы изменения в ней не затронули исходные данные.
* `len()` — вычисляет общее количество строк в таблице или её фрагменте.

3. ### Добавление новых столбцов  
* Создание производного столбца `df['new_col']` — инициализирует новый столбец в структуре данных и заполняет его значениями на основе вычислений.
* Векторизованная операция (`*`) — выполняет поэлементное умножение значений из двух столбцов для всех строк одновременно без использования циклов.
* Арифметическое выражение (`1 - df['discount']`) — позволяет применить математическую формулу к каждому значению в столбце, автоматически создавая промежуточный массив данных для расчетов.
* Оператор сравнения `==` — сравнивает каждое значение в выбранном столбце с эталоном и возвращает логическое значение.
* Логический тип данных (`bool`) — автоматически назначается столбцу `is_valid_sale`. Этот тип данных наиболее эффективен с точки зрения памяти и скорости фильтрации.
* Выборка списка столбцов `df[['col1', 'col2']]` — метод «двойных квадратных скобок», который позволяет временно отобразить или скопировать только нужные части таблицы.

4. ### Бизнес-анализ  
`total_net_income = df[df['status'] == 'Completed']['net'].sum()`  
- `df['status'] == 'Completed'` - выдает `True` (если это `Completed`) или `False`  
- `sum()` - математическая функция, которая складывает все числа в полученном столбце `net`.

`top_products = df.groupby('product')['net'].sum().sort_values(ascending=False).head(5)`  
- `groupby('столбец')` — группирует данные по уникальным значениям в выбранном столбце.
- `['столбец']` — ограничивает область расчетов конкретным показателем.
- `.sum()` — суммирует значения внутри каждой группы.
- `.sort_values()` — сортирует данные (в данном случае для выявления лидеров продаж).
- `.head(n)` — возвращает первые n записей результата.  

`category_analysis = df[df['status'] == 'Completed'].groupby('category').agg({
    'net': 'sum',
    'quantity': 'sum',
    'discount': 'mean'
})`  
- `.groupby('category')` — группирует записи по категориям товаров для проведения сравнительного анализа.
- `.agg()` — метод агрегации, позволяющий вычислять несколько разных метрик (сумму, среднее и др.) для разных столбцов за один шаг
- `mean` — функция для нахождения среднего значения

`status_counts = df.groupby(['country', 'status']).size().unstack(fill_value=0)`  
- `df.groupby(['col1', 'col2'])` — создает иерархическую группировку по нескольким признакам одновременно.
- `.size()` — вычисляет количество записей (строк) для каждой уникальной комбинации группы.
- `.unstack()` — переносит уровень индекса строк в столбцы, преобразуя структуру таблицы для удобства сравнения показателей.
- `fill_value=0` — заполняет пропущенные данные нулями, предотвращая ошибки в последующих математических расчетах.

`status_counts['return_rate'] = status_counts.get('Returned', 0) / (status_counts.get('Completed', 0) + status_counts.get('Returned', 0))`
- `df['new_col']` — создает новый расчетный показатель в сводной таблице.
- `.get(key, default)` — безопасно извлекает данные по имени столбца. Если данные отсутствуют, подставляет 0, предотвращая поломку кода.  

5. ### Итоговый отчет
- `resample('ME', on='...')` — изменяет частоту временных данных, группируя их по заданному периоду (месяц, неделя, год).
- `on='order_date'` — задает столбец с датами, который будет служить основой для временной группировки.
- `.idxmax()` — возвращает индекс (название строки или категории), соответствующий максимальному значению в столбце.
- Формат :`.2f` — округляет числовое значение до двух знаков после запятой, приводя данные к денежному виду.

### График  
- `kind='bar'` - тип графика (столбчатая диаграмма)
- `figsize=(10, 6)` - Устанавливает размер окна графика в дюймах  
- `plt.title(...)`- Добавляет главный заголовок сверху
- `plt.xlabel(...)` и `plt.ylabel(...)` - Подписывают горизонтальную (`X`) и вертикальную (`Y`) оси  
- `plt.xticks(rotation=45)` - Поворачивает названия категорий под углом 45°
- `plt.grid(axis='y', ...)` - Включает горизонтальную сетку. Параметр alpha=0.7 делает её полупрозрачной
- `plt.tight_layout()` - Автоматически подбирает отступы
- `plt.savefig('income_by_category.png')` - Физически создает файл изображения на диске
- `plt.show()` - Выводит готовый график на экран в интерактивном окне
