### Выполнение лабораторной работы  
1. #### Рассчитать возраст в днях  
**Модуль `datetime` позволяет:**
- получать текущую дату и время;
- создавать даты вручную;
- выполнять вычисления с датами;
- считать разницу между датами;
- форматировать дату и время. 
   
**Модуль `calendar` работает именно с календарём:**
- показывает календарь месяца;
- определяет день недели;
- проверяет високосный год;
- считает количество дней в месяце.

- Функция `user_birthday()` запрашивает у пользователя дату рождения, проверяет правильность формата и корректность 
введённой даты (месяц и количество дней в месяце), после чего возвращает её как объект `datetime.date`. При ошибке ввода 
пользователь должен ввести дату повторно.  

- Функция `lived_days(human_birthday)` вычисляет количество дней, прожитых пользователем. Она получает текущую дату, 
проверяет, что дата рождения не находится в будущем, после чего определяет разницу между текущей датой и датой рождения 
и выводит количество прожитых дней.  

Результат вывода функции:
```
Enter your birthday: 17.11.2004
Lived days:  7831
```
2. #### Определение дня недели для определенной даты  
- Функция `get_date() запрашивает` у пользователя ввод даты, проверяет её формат (дд-мм-гггг или дд.мм.гггг) с помощью 
регулярного выражения и извлекает день, месяц и год. Затем выполняется проверка корректности даты через функцию
`calendar.weekday()`. Если дата введена правильно, функция возвращает год, месяц и день, иначе выводит сообщение об 
ошибке и повторяет ввод.  

- Функция `get_weekday()` определяет день недели по введённой дате. Она использует функцию `calendar.weekday()` для 
получения номера дня недели, после чего возвращает его название на русском языке из заранее созданного списка дней недели.  

Результат вывода функции:
```
Введите дату: 23.02.2000
Дата 23-02-2000 приходится на: Среда
```  
3. Расчет времени падения объекта  
- Функция `get_height()` запрашивает у пользователя высоту падения в метрах и проверяет корректность ввода. Она повторяет 
ввод, пока пользователь не введёт положительное число. Если введено не число или отрицательное значение, выводится 
сообщение об ошибке. После успешной проверки функция возвращает введённую высоту как число типа `float`.
- Функция `calculate_fall_time(h)` вычисляет время падения тела с высоты `h`.  

Результат вывода функции:
```
Введите высоту падения в метрах: 23
Время падения объекта: 2.17 секунд
```  
4. Секундомер. Библиотека TkInter
Программа создаёт секундомер с графическим интерфейсом `Tkinter`. Она позволяет запускать, останавливать, продолжать и 
сбрасывать отсчёт времени. Время считается в секундах и отображается в формате `MM:SS`. Таймер обновляется каждую 
секунду с помощью функции `after()`, а управление осуществляется кнопками «Старт», «Стоп», «Продолжить» и «Сброс».
